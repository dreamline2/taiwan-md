# 日本所藏臺灣近代政治社會運動資料蒐集及翻譯整理計畫

- Plan ID: `0e501608-203e-468c-a2cd-86855e53afdb`
- Plan Number: I-J-009
- Source Type: 結案報告書
- Status: Published
- Source: https://taiwanoverseas.nmth.gov.tw/archives/plan/0e501608-203e-468c-a2cd-86855e53afdb
- Continents: 亞洲

## Description

本計畫係針對日本典藏機構所藏日治時期臺灣社會運動相關史料進行調查，包含労働図書館的所藏社會運動通信、國會圖書館所藏戰前社會運動刊物與書籍、法政大學大原社會問題研究所所藏檔案、外務省外交資料館所藏不逞団關係雑件台湾人の部，共完成史料解題 15 篇，史料編目 677 件，其中翻譯 261件，達 63 萬字。因為內容量多，故分為兩冊，上冊含労働図書館的所藏社會運動通信、國會圖書館所藏戰前社會運動刊物與書籍。下冊含法政大學大原社會問題研究所所藏檔案、外務省外交資料館所藏不逞団關係雑件台湾人の部。

完成年份：2019年
