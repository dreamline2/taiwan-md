# 說明

此 PR 解決了翻譯文章（韓文、英文、日文）中註腳無法正確渲染的問題，並同步強化了「台灣民歌運動」文章在各語系中的學術查核內容。

## 關鍵變更

### 1. 註腳渲染系統修復

- 在 `src/pages/[lang]/[category]/[slug].astro` 中實作了手動註腳解析器，支援 `marked` 渲染流程中的 GFM 註腳語法 (`[^1]`)。
- 移植了中文版的精美註腳 CSS 樣式，包含藍色上標、跳轉高亮以及行動端優化。
- 修正了註腳跳轉的偏移量（scroll-margin-top），確保跳轉後內容不會被 Header 擋住。

### 2. 內容強化（民歌運動）

- 為《台灣民歌運動》文章新增了 9 個可驗證的腳註，涵蓋胡德夫、羅大佑、《龍的傳人》創作背景等關鍵歷史事實。
- 同步了繁體中文與韓文版本的內容補強。

### 3. Slug 與翻譯路徑標準化

- 將所有語系的檔案名稱統一為英文 Slug (`taiwan-campus-folk-song-movement.md`)。
- 更新了 `knowledge/_translations.json` 對應表，修復了跨語系連結產生的 404 錯誤。

## 驗證項目

- [x] 驗證反向跳轉箭頭 (↩) 功能正常。
- [x] 驗證修改 Slug 後，跨語系切換不再出現 404。
- [x] 視覺對比驗證：與《鄭麗文》參考文章的樣式一致。
