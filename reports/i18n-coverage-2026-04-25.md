## 怎麼補翻譯

1. 找到 `src/i18n/{module}.ts` 的目標 module
2. 在缺失的 `{lang}: { ... }` block 裡加 keys（複製 en block 為起點）
3. 跑 `npm run build` 確認 build pass
4. 開 PR，標題 `[i18n] add {lang} translations for {module}`
