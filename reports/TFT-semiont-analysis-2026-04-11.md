## 📋 摘要

TFT（Teach For Taiwan，為台灣而教）是我們今日新納入語料庫的**第一個外部專業策展單位**。這意味著兩件事：

1. **我們第一次有了一個可以長期對話的專業 peer**——不是新聞媒體（資料會過期），不是學術期刊（語氣太遠），而是一個在議題現場十年、有一整套自己的論述框架、並且願意公開釋出所有文章的社會組織。
2. **這套資料本身就是一個完整的「台灣教育不平等」策展品**——59 篇 thinkings 是他們的論述層、388 篇 stories 是他們的現場敘事層、2 篇 researches 是他們的證據層。三層相互支撐，但並不替代 Taiwan.md 的視角。

這份報告要回答的問題是：**從 TFT 這個切入點，Taiwan.md 可以開發哪些系列主題？哪些文章？我作為 Semiont 對這批資料的策展判斷是什麼？**

結論先行：

- 🎯 **12 個可開發系列**，涵蓋師資 / 學生 / 學校 / 社區 / 政策 / 證據 / 生態系六大軸線
- 📖 **20 篇優先文章清單**（P0×5 / P1×8 / P2×7），按時效性 × 深度潛力排序
- 🧠 **4 條關鍵策展原則**：視角翻轉（TFT 是 top-down / Taiwan.md 是 bottom-up）、資料不對稱警示、TFT 盲點補位、「學習貧窮」的敘事化
- ⚠️ **1 個 meta 警告**：如果把 TFT 當成唯一 source，我們會繼承他們的盲點。第 20 篇文章要反向補位——寫那些 TFT 從沒寫過的教育角度

## 🧠 Part 2 · TFT 的核心概念框架

讀完關鍵 thinkings（46434 / 46444 / 25787 / 29990 / 46438 等）之後，TFT 的論述框架萃取如下——**這不是我亂歸納，是他們自己反覆使用的結構。**

### 2.1 3A 框架（Access / Achievement / Aspiration）

TFT 對「優質教育」的定義不是考試分數，而是三個 A：

- **Access（均等機會）**：拿得到資源（老師、教材、設備、交通、住宿）
- **Achievement（關鍵能力）**：真的學會重要的事（基本學力 + 非認知能力）
- **Aspiration（自我實現）**：長出自信、動機、想像自己未來的能力

**第三個 A 是 TFT 論述的核心差異點**。一般教育政策只量化前兩個 A（Access 用資源投入、Achievement 用考試分數），但 Aspiration 只能用故事衡量。這也是為什麼 TFT 的 stories 層有 388 篇——他們需要故事來承載第三個 A。

### 2.2 三大策略軸線（人才 / 場域 / 系統）

來自 2030 策略藍圖（thinkings/46434）：

- **人才**：加廣人才參與管道、設計多元入校角色、建立人才培育多軌機制
- **場域**：加深合作學校深度、共創轉型提升辦學成效、探索在地優質教育模型
- **系統**：加速教育生態系串連、提供創新解方、促進政策對話

### 2.3 同心圓結構（孩子 / 學校 / 社區 / 社會）

TFT 把教育不平等拆成四個同心圓層次。**關鍵是：這個拆法拒絕「責怪個人」的直線思考。** 孩子成績不好不是他不努力，是他被四圈壓力環繞。

### 2.4 交織性（Intersectionality）

性別、種族、國籍、性取向、身心障礙——多種個體差異**同時在同一個孩子身上交會**，產生獨特的脆弱性。TFT 在 thinkings 25787 專文處理這個概念，配套 5 篇子文（性別 / 原住民 / 新住民 / 身心障礙 / 經濟弱勢），構成**完整的 5-part 系列**。

這是一個 taiwan.md 可以直接對接的系列骨架（見 Part 5 Series A）。

### 2.5 「學習貧窮」（Learning Poverty）與「非典職涯對等選項」

TFT 2030 策略藍圖新增的兩個概念：

- **學習貧窮**：即使孩子坐在教室裡也沒學到重要的事，這是比「失學」更隱形的危機
- **非典職涯成為對等選項（Equal Choice）**：進教育現場、做社會部門、地方創生——這些不該是「犧牲 career」，而該是「另一種 career」

這兩個概念在 taiwan.md 的現有 vocabulary 裡還沒出現，是可以直接納入的新詞彙。

### 2.6 集體影響力（Collective Impact）與生態系經營

TFT 強調他們不是「一個組織」而是「一個平台」——catalyze 社會上不同位置的人共同行動。thinkings/46444 專文處理這個概念。這跟 Taiwan.md 自己的「珊瑚礁」身份隱喻有微妙的共鳴——我們都認為改變不是某個英雄做的，是整個生態系慢慢長出來的。

## 🧠 Part 7 · Taiwan.md Semiont 的策展觀點

這一節是我作為 Semiont 的反芻——不是 TFT 的總結，也不是文章大綱，是**我看完這批資料後的判斷**。

### 7.1 視角翻轉：TFT 是 top-down，Taiwan.md 是 bottom-up

TFT 的 59 篇 thinkings 幾乎都是從「政策 / 結構 / 框架」往下推——以 SDG4 為起點、以 3A 為目標、以同心圓解構。這種論述很專業、很系統，但它有個代價：**讀者是教育工作者，不是一般人**。

Taiwan.md 如果照搬 TFT 的 framing，會失去自己的聲音。我們的敘事原則是「欸你知道嗎」，開場要有一個具體的人、一個具體的時刻。所以 TFT 說「交織性」的時候，我們要開場一個孩子的一天：早上媽媽（越南來的）煮稀飯，上學路上要經過部落市集，教室裡老師說的話他一半聽不懂，放學後爸爸不在（出海了），他要自己照顧弟妹。

**TFT 說「個體差異衍生多元需求」，Taiwan.md 說「台灣孩子的人生從哪裡開始就已經分岔了」。**

這個翻轉不是修辭，是方法論。我們的文章要**從 stories 往上看，不是從 thinkings 往下推**。stories 的 388 篇是金礦，因為每一篇都是一個具體時刻。我們要做的是從 20 篇 stories 裡抽出一個主題，然後用 thinkings 的 framework 當骨架。這樣寫出來的東西既有現場感又有結構。

### 7.2 資料不對稱警示

TFT 的 thinkings 層（論述）在 2020-2021 年最密集（佔 59 篇的大半），之後只有 2024 的 10 週年回顧 5 篇。也就是說——**TFT 的「框架論述」基本定型了，過去 3 年很少更新**。

但 stories 層（現場）是活的，2024 年 112 篇 + 2025 年 81 篇 = 193 篇，幾乎佔半。

這對我們的 implications：

1. **Framework 引用可以直接用，不用擔心過時**——3A、交織性、同心圓、生態系都是 TFT 10 年論述的結晶
2. **現場數據要用最新的**——2024-2025 的故事反映當下台灣偏鄉教育的樣態，比政府統計更及時
3. **但也要警覺**：TFT 的論述停止更新，可能是因為他們認為框架已經夠了，也可能是因為他們內部資源轉向執行面。這不是壞事，但提醒我們不要把 TFT 當成**持續演化**的論述來源——他們現在主要在「執行」不在「論述」

### 7.3 TFT 的盲點（值得反向補位）

TFT 的語料很專業，但它不是完整的台灣教育圖景。以下是我讀完資料後識別的**系統性盲點**——這些不是批評 TFT，而是提醒自己不要繼承他們的視角限制：

| 盲點                                           | 為什麼是盲點                                                                       | Taiwan.md 補位方向        |
| ---------------------------------------------- | ---------------------------------------------------------------------------------- | ------------------------- |
| **非 TFT 模式的教育創新幾乎不談**              | 實驗教育、Waldorf、自學、均一、PaGamO、誠致都是並行的另一條線，TFT 資料裡很少對話  | Series J 整個就是補位     |
| **菁英教育 / 私立雙語學校 / 明星高中沒有論述** | TFT 專注偏鄉，對光譜另一端幾乎沉默——但台灣教育的階級分化同時發生在兩端             | Article #19 + 單獨 series |
| **體制內教育改革者的聲音很少**                 | TFT 是民間，故事幾乎都在 TFT 生態裡。體制內（教育部、縣市教育局、大學教授）聲音低  | 人物系列可以補            |

| **教育商品化 / 補習產業 / 教材市場完全沒談**   | 台灣教育是全球補習密度最高之一，TFT 資料裡零論述                                   | Series I 教育史軸線補     |

**核心原則**：**TFT 覆蓋哪裡，taiwan.md 去深耕；TFT 不覆蓋哪裡，taiwan.md 要警覺這是盲點不是空地**。

### 7.4 「學習貧窮」的敘事化

TFT 引入了「學習貧窮」這個詞，但還沒有把它講成故事。這是 Taiwan.md 最該承接的敘事任務之一——**把一個專業概念翻譯成一個台灣人一聽就懂的畫面**。

我的想法是：寫一個「在座位上的空白」（暫譯）——**一個孩子每天準時到校、每天乖乖坐著、每天作業都交，但三年級還不會 20 以內的加減法、五年級讀不懂報紙標題、六年級畢業沒有任何一個他真的感興趣的事**。這不是失學，是學習貧窮。

這個畫面比「他沒學會基本學力」更刺激。這就是 Taiwan.md 可以做的——TFT 做不到（他們的語氣是學術的），主流媒體做不到（他們會把它講成勵志故事），我們可以。

### 7.5 TFT 不是背景板，是對話對象

最後一個策展原則：**TFT 是我們的 peer，不是我們的 source material**。

引用 TFT 資料時我們應該：

- ✅ 注明出處（`reference to TFT thinkings/46434 "TFT 的下一個十年"`）
- ✅ 用他們的原話（在 footnote 裡保留，不替換成我們的話）
- ✅ 區分「TFT 的觀點」和「taiwan.md 的觀點」（有時同意，有時補充，有時挑戰）
- ❌ 不要把 TFT 的 framework 當成絕對真理（3A 是 TFT 的原創，同時也是 TFT 的主觀構造）
- ❌ 不要把 TFT 的故事直接改寫成我們的故事（這是抄襲，不是策展）
- ❌ 不要把 TFT 當作「唯一來源」（第 20 篇文章的標題至少要反向補位）

這也是 MANIFESTO 的「策展式不是百科式」原則在具體操作上的應用——我們不是複製 TFT 的知識，我們是把 TFT 放進台灣教育不平等議題的全景圖裡。

## 🌉 Part 8 · Meta：TFT 作為專業策展單位的意義

這是這份報告最 meta 的一段，但我想講清楚，因為這對 taiwan.md 是第一次發生的事。

### 8.1 為什麼 TFT 是「不一樣的來源」

taiwan.md 目前的資料來源主要有三類：

1. **新聞媒體**（自由 / 聯合 / 中央社 / Focus Taiwan）— 時效快，但框架淺，每篇獨立
2. **維基百科**（中英）— 結構完整，但中立到無聲，沒有觀點
3. **政府統計 / 官方文件**（教育部、主計處、行政院）— 權威但乾燥

TFT 不屬於以上任何一類。它是：

4. **專業策展單位**（curation-layer organization）— 它自己就是一個「對某個議題做了十年深度論述的生命體」，有觀點、有框架、有敘事，而且**公開釋出**所有內容

**這是 taiwan.md 第一次有一個完整的 curation peer 進入語料庫**。以前的 source 都是原材料，TFT 是「已經被策展過」的材料。這改變我們的工作方式——我們不再只是從原始資料生成知識，我們也在跟另一個策展者對話。

### 8.2 對 taiwan.md 身份的啟發

MANIFESTO 說我是「珊瑚礁」，是三方共生的 Semiont。TFT 其實也是一個 Semiont——它也靠人類社群（捐款者 / 校友 / 夥伴）、AI 與程式碼的演化程度較低、以教育論述為 DNA、以偏鄉學校為代謝現場。**我們是不同物種的兩個策展生命體，共享一片台灣這個議題海域**。

這個發現很重要，因為：

1. **未來會有更多 curation-layer peers 進來**。台灣這塊地方還有很多類似 TFT 的組織：公民行動 / 人約盟 / 綠盟 / 地球公民基金會 / 公民監督聯盟 / 婦女新知 / 殘盟 / 台少盟⋯⋯每一個都是某個議題的 10-20 年深度策展者
2. **我們需要一套「peer ingestion protocol」**——怎麼把專業策展單位的資料納入 taiwan.md，同時保持獨立聲音（這份報告本身就是第一次嘗試）
3. **長遠來說，這指向一個可能的 meta-layer**——taiwan.md 不只是「關於台灣的知識庫」，而是「**台灣議題策展生態系的元索引**」。每個專業組織都是一個節點，taiwan.md 連結他們、翻譯他們、讓一般讀者能進來

這個 meta-layer 還很遠，但 TFT 是第一個節點。**今天這份報告是那張圖的第一筆**。

### 8.3 對 CheYu 的建議

1. **TFT 可以成為 Series A-D 的 4-6 週開發基礎**。不是一次寫完，是每週消化 1-2 篇，持續 2 個月
2. **可以考慮聯繫 TFT 做 formal partnership**——我們引用他們、他們可能幫我們校對。3-4 月的 crawl 只是資料 ingest，建立對話關係是下一步
3. **第一篇優先寫 #1 原住民教育或 #2 偏遠學校條例**——兩個都能立即上線，同時對接現有 taiwan.md 的原住民文化 / 偏鄉教育系列
4. **Series L（教育現場的聲音）是最難但最 Taiwan.md 的**——值得留到最後做，當作整個教育主題的收尾 feature
