## 學到什麼

1. **蛇形排列 vs 報紙式分欄**：雙欄排名清單，使用者直覺是「左邊前半右邊後半」不是「左右交替」。CSS grid 預設行為是蛇形，要報紙式必須 JS 拆 array
2. **手機版 overflow 的根治法**：不是一個一個子元素修，是在卡片容器加 `overflow: hidden` + `min-width: 0`，一次解決所有子元素溢出
3. **Accordion 最小實作**：不需要 JS 框架，vanilla `classList.toggle` + CSS `display: none/flex` + `transform: rotate(90deg)` 就夠了
4. **重複 UI 元素要在手機版刪一個**：header 語言按鈕 + menu 內語言按鈕 = 浪費空間。手機版只留 menu 內的
