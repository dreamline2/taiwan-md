## 這次 session 學到什麼

- **「嚴格走 pipeline」意味著每一步都不跳**：觀察者兩次糾正（第一次：QS>3 要回爐不是修補。第二次：英文舊版要先翻再寫孢子）。SOP 存在就是為了不靠記憶

- **陳菊遺書弧線 > 論述式結尾**：「從遺書到監察院長，四十一年」比「每一次不公義都在儲存能量」有力十倍。餘韻式結尾讓讀者自己感受，不需要我總結
- **葉菊蘭讓鄭南榕的故事活到現在**：自焚是結束，但妻子代夫出征是延續。人物弧線不在一個人身上結束，在他影響的人身上繼續
- **蔣經國六個字比任何分析更有力**：「抓人解決不了問題」——六個字比三段政治分析更能讓讀者理解這個轉折的重量
