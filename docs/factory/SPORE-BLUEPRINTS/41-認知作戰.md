### Angle 選定 → **Angle B**（沈伯洋「敵人勳章」）

選擇理由：

- **時效性**：2025-10-28 立案（距離今天 < 6 個月），央視 8 分鐘專題是新聞級大事件
- **人物 hook**：沈伯洋是立委 + 黑熊學院創辦人 + IORG 共同主持，故事張力夠強
- **諷刺張力**：教別人防認知作戰的人被中共認證 → 這個諷刺本身就是孢子金句
- **避開敏感**：不用 Angle C 的關西機場（蘇啟誠自殺有道德雷區）
- **政治溫度**：中共立案是既成事實、國安局處理，非綠營發起議題，政治攻擊性低

### Angle B 核心素材

1. **Scene hook**：2025-10-28 重慶市公安局警情通報 → 「分裂國家罪」立案偵查
2. **Quote 1（沈伯洋）**：「一邊一國，別妄想手伸進台灣」
3. **背景 pivot**：黑熊學院 2021 + 曹興誠 6 億 = 教台灣人認知戰的人
4. **反轉 twist**：被中共用國家機器認證工作價值 = **敵人勳章**
5. **ending hook**：黃澎孝那句「恭喜...獲頒《敵人勳章》」

### 算術

- 2025 - 2021 = 4（黑熊學院創立至立案 4 年）
- 2024-10-14 → 2025-10-28 = 約 1 年（國台辦清單 → 公安立案）

### 敏感度標記 → 觸發 Step 2.7

- **政治敏感高**：涉及中共國家機器、台灣政治人物（但非在任職涉入）
- **避開雷區**：不提關西機場 / 蘇啟誠（Angle C 棄用）
- **處理方式**：
  - 用「中共認證工作價值」的諷刺而非「中共罪惡」的指控
  - 央視「全球抓捕」明確 framing 為節目言論非正式通緝令
  - 對沈伯洋本人：保留他的「學者 + 立委 + 民防建構者」三重身份，避免塑造為「英雄」

### 四問自檢（高敏感場景強制）

1. **主角被同意了嗎？** → 沈伯洋是公眾人物 + 這些都是他公開回應的事件（回應央視、接受採訪），沒隱私爭議 ✅
2. **有沒有編造情感或心理狀態？** → 不寫「他一定很害怕」「他一定很驕傲」這類 internal 推測 ✅
3. **引語是他真的說過的嗎？** → 「一邊一國」有中文新聞原話對齊 ✅
4. **讀者讀完後的情緒朝向**？ → 期待引發「這個諷刺好有張力」的思考，而非「來肉搜中共官員」的敵意 ✅
