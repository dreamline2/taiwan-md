## Step 2.7 倫理閘（針對死亡相關 bullets）

**四問自檢**：

1. **是否以死亡作為情感鉤？** — 原初 draft 確實過度依賴凡凡的死亡。把它抽掉，故事還剩「2017 vs 2024 兩次獎」=**架構會崩**。⚠️ 觸發 angle 重組。
2. **細節是否必要？** — #5 / #6 / #7 / #8（10-25 入住 / IG 原文 / 15 小時 / 土豆病危）屬**獵奇重構**，不是敘事推進。✗ 全部刪。
3. **主體自己的公開行動是什麼？** — 樂團做了四個公開追悼姿態：
   - 《瓦合》扉頁「謹以此專輯獻給凡凡」
   - 〈但〉(Damn) 最後一首吶喊「我愛你」
   - 經紀人代領「不只是 4 個人」
   - 2024 每首 MV 推廣「致我們最可愛的凡凡」
     → **這些是主體公開 framing，可以寫**。
4. **活下來的人在做什麼？** — 他們選了一個《史記》裡失敗者的詞「瓦合」來命名第二張專輯，然後在缺席金曲典禮的情況下得獎。**這是 agency，是故事的重心**。

### 決策：

- **刪除**：#5 / #6 / #7 / #8（death-scene reconstruction）
- **保留**：#4（過世年份 + 身份），但簡化為「2021 年過世」，不重構現場
- **強化**：#10（扉頁獻詞）作為主體公開 framing 代表；《瓦合》典故作為智性/文學 hook 取代 death hook
- **新加**：現場「安靜了五秒」（原文結尾）作為集體見證描述，取代個人死亡細節
